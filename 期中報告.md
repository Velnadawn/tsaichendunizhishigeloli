## 前言
MATLAB是由美國The MathWorks公司出品的商業數學軟體。MATLAB是一種用於演算法開發、資料視覺化、資料分析以及數值計算的進階技術計算語言和互動式環境。除矩陣運算、繪製函數/資料圖像等常用功能外，MATLAB還可用來建立使用者介面，以及呼叫其它語言編寫的程式。
## 主題一：CH08二維圖形
### 大綱        
二維圖形可以簡單的看出數據間的關係，譬如最常見的線性圖 </br>
plot - 進行二維平面的描點，把點連起來 </br>
grid on/off - 顯示和隱藏座標方格, xlabel/ylabel - 標示x&y軸名稱</br>
text - 在(x,y)座標上標示文字, gtext - 互動式文字</br>

box on/off - 圖表邊框</br>
fill - 顏色, area - 區域填色</br>
subplot() - 在一個顯示框中生成多個圖形(二維&三維都能用)</br>
### 實作
- 課程練習: 
多重圖形(多個圖形在同個介面)
```matlab
%subplot(m,n,p)
subplot(2,2,1)
subplot(2,2,2)
subplot(2,2,3)
subplot(2,2,4)

x=linspace(0,4*pi,100);
y1=sin(x);
subplot(2,2,1);
plot(x,y1);
axis tight;

%%
y2=cos(x);
subplot(2,2,2);
plot(x,y2);
axis tight;

%%
y3=tan(x);
subplot(2,2,3);
plot(x,y3);
axis([0,4*pi,-10,10]);
```
說明:先分割出要用到的位置，接著將x從0~4pi分割成100個點，生成sin(x),cos(x),tan(x)

- 自己想的主題:
在GUI中調整圖片解析度(CH16動畫+CH07握把與GUI介面)
function image_resolution_gui_fixed
```matlab
   %建立 GUI 視窗
   f = figure('Name','圖片解析度調整器', ...
              'Position',[400 200 600 450]);
   %載入圖片
   img = imread('peppers.png');
   hAxes = axes('Parent', f, 'Position', [0.1 0.2 0.8 0.7]);
   imshow(img, 'Parent', hAxes);
   title('原始圖片');
   %記錄目前縮放倍率
   scaleFactor = 1;
   %提高解析度按鈕
   uicontrol('Style', 'pushbutton', ...
             'String', '提高解析度', ...
             'FontSize', 12, ...
             'Position', [150 40 120 40], ...
             'Callback', @increase_resolution);
   %降低解析度按鈕
   uicontrol('Style', 'pushbutton', ...
             'String', '降低解析度', ...
             'FontSize', 12, ...
             'Position', [330 40 120 40], ...
             'Callback', @decrease_resolution);
   %放大函式
   function increase_resolution(~,~)
       scaleFactor = scaleFactor * 1.5;  % 調整倍率
       resizedImg = imresize(img, scaleFactor, 'bicubic'); % 每次從原圖放大
       imshow(resizedImg, 'Parent', hAxes);
       title(['解析度提高至 ' num2str(scaleFactor, '%.2f') ' 倍']);
   end
   %縮小函式
   function decrease_resolution(~,~)
       scaleFactor = scaleFactor / 1.5;
       resizedImg = imresize(img, scaleFactor, 'bicubic'); % 每次從原圖縮小
       imshow(resizedImg, 'Parent', hAxes);
       title(['解析度降低至 ' num2str(scaleFactor, '%.2f') ' 倍']);
   end
end
```
說明:主要使用imresize()調整解析度。
- 心得
	這堂課中我學到了各種改變圖片的程式碼，像是將圖片變成灰階，增加減少解析度，以及數位影像點處理。這個增減解析度的實作是可以在需要更清晰碩更模糊的圖片時使用，像是太血腥的圖片可以打碼，太糊的風景照可以變更清晰。

## 主題三：CH09三維圖形                                                                               
### 大綱 
將資訊靠圖形的技巧來呈現，其中三維空間的立體繪圖是一個非常基本且重要的技巧，可生成線條、網格、表面圖、觀測點、等高線圖、特殊圖，以便觀察數據間的關係。</br>

- 自己想出的主題:
彩色糖果
```matlab
% 原本橫放的橢球
[X, Y, Z] = ellipsoid(0,0,0,15,5,10,80);

% 旋轉矩陣：繞 Y 軸轉 -90 度（讓長軸改成沿 Z 軸）
theta = -pi/2;
R = [cos(theta) 0 sin(theta);
     0          1 0;
    -sin(theta) 0 cos(theta)];

pts = [X(:), Y(:), Z(:)]';
rot = R * pts;
Xr = reshape(rot(1,:), size(X));
Yr = reshape(rot(2,:), size(Y));
Zr = reshape(rot(3,:), size(Z));

% 繪圖
surf(Xr, Yr, Zr, Zr, 'EdgeColor', 'none');
colormap(jet);
colorbar;
axis equal; hold on;
set(gca,'color','k');
lighting gouraud; camlight headlight;
title('旋轉後的垂直彩色橢球','Color','w','FontSize',14);
```
說明:使用ellipsoid()生成橢圓形，用旋轉矩陣把橢圓直立，用colormap(jet)變成彩虹色
